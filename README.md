# Movie Management System (Java + JDBC + MySQL)

This is a console-based Movie Management System developed using Java and JDBC with MySQL database.

## Features

- User Registration
- User Login Authentication
- Add Movie
- View Movies with Pagination
- Search Movie
- Update Movie Rating
- Delete Movie
- Sort Movies
- Filter Movies

## Technologies Used

- Java
- JDBC
- MySQL
- DAO Design Pattern

## Database Tables

### users table
- id
- username
- password

### movies table
- id
- name
- genre
- rating

## How to Run

1. Create MySQL database:
2. Create tables

3. Update db.properties file

4. Run UserMain.java
